### webpack学习笔记

#### 1. loader使用方法  
  * webpack.config.js中设置某一类后缀名的文件使用相对应的loader  
  * require代码中指定相应的loader

##### 2. loader类型  
| 名称 | 作用 |  
| ---- | --- |
| css-loader | js中加载css文件的loader | 
| style-loader | js中应用css的loader |
